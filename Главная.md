# Голосовой дневник

Сюда автоматически попадают голосовые заметки, отправленные телеграм-боту.

## Как это устроено

1. Отправляешь войс (или текст) боту в Telegram — в любой момент, с телефона.
2. Бот на VPS сохраняет сообщение и отвечает ✅.
3. Когда комп включён, фоновый скрипт каждые 10 минут забирает новые сообщения,
   расшифровывает голос через Whisper (локально, офлайн) и прогоняет через Claude:
   чистит оговорки, делает саммари, ставит теги.
4. Заметка попадает в папку [Голосовые](Голосовые) — одна заметка на день,
   записи внутри по времени. Аудио-оригиналы лежат в `Вложения/аудио` и
   прикреплены к каждой записи.

## Структура записи

- **Суть** — саммари в одну строку (удобно сканировать глазами список за месяц).
- **Теги** — `#темы` для поиска и анализа через годы.
- Очищенный текст — читабельная версия сказанного.
- Сырой транскрипт — свёрнут, сохраняется дословно на случай, если Claude что-то исказил.

## Анализ через годы

Все записи — обычный markdown с датами, тегами и frontmatter, так что можно:
- искать по тегам и словам средствами Obsidian;
- смотреть граф связей;
- скормить папку `Голосовые` Claude и попросить найти закономерности за любой период.
